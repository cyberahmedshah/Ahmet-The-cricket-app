# Ahmet- "The Cricket App"

Ahmet is my first Full App. It is a cricket management application designed to simplify player statistics and tournament administration.

## Features

* Calculate batting average and strike rate
* Calculate bowling average and strike rate
* Manage tournaments and team records
* Manage points tables
* Calculate Net Run Rate (NRR)
* Track team and player performance

## Current Development

* Implementing the Duckworth-Lewis-Stern (DLS) method for rain-affected matches

## Goal

This project aims to provide an easy-to-use tool for cricket players, organizers, and enthusiasts to manage statistics and tournaments efficiently.
